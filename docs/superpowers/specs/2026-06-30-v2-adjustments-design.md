# 生日卡片 v2 調整 — Design Spec
Date: 2026-06-30

## 背景
v1 已上線（https://tianw1998.github.io/birthday-card-2026-dad/）。使用者手機實測後提出 10 項調整＋美術風格全面升級＋祝福文字替換。採四階段執行，每階段部署後由使用者驗收再繼續。

## 階段 1：快修（一次做完直接部署）
| 項目 | 變更 | 檔案 |
|------|------|------|
| 右邊界 bug | 修正碰撞/clamp，爸爸不可走出地圖右側 | `js/config.js` 或 `js/player.js` |
| 移速 | `PLAYER_SPEED` 2.5 → 1.5 | `js/config.js` |
| 標題 | 「🎂 給爸爸的生日卡片　」（含結尾全形空格） | `index.html` |
| 副標 | 標題下方加「操控下方搖桿來到家人旁邊吧💕」小字 | `index.html` + `style.css` |
| 對話按鈕 | 移除 [關閉]，只留 [接收] | `index.html` + `js/dialog.js` |
| 祝福文字 | 替換為新版（見下） | `js/config.js` |
| 手機縮放 | canvas 依螢幕尺寸等比縮放地圖（`scale = min(cw/mapW, ch/mapH)`），點擊座標同步換算；此邏輯與地圖尺寸無關，直式地圖沿用 | `js/renderer.js` + `js/main.js` |

### 新版祝福文字
- 媽媽：「親口對您說！」
- 哥哥：「親愛的爸爸，感謝您總是為家庭遮風擋雨，做我們最堅強的後盾。祝您生日快樂，天天開心，身體健康，永遠是我們心中最偉大的英雄！」
- 弟弟：「在這個重要的日子，格外想念您，您給予的愛與關照、有您相伴的成長回憶也湧上心頭，祝爸爸生日快樂！」

## 階段 2：BGM 試聽
- 找 2–3 首 CC0 溫馨輕快鋼琴曲（FreePD / Pixabay）
- 部署 `bgm-test.html` 試聽頁（手機點播），使用者選定後替換 `assets/bgm.mp3`，刪除試聽頁

## 階段 3：美術重做（核心）
### 素材
- LimeZu「Modern Interiors」免費版（itch.io），非商業使用，頁尾標註「Tiles by LimeZu」（v1 已有）
- 裁出所需 tile 合成單一 sprite 圖集 `assets/atlas.png`，`drawImage` 渲染，`imageSmoothingEnabled = false`
- 備案：下載不順則退回程式手繪精細像素，另行討論

### 直式地圖（約 10×13 格）
- 木頭沙發貼底牆：媽｜弟｜哥 並排（弟在中間）
- 右側沙發改直向靠右牆
- 中央走道空曠
- 電視牆、書櫃、蛋糕長桌、藤椅、盆栽、門沿用原格局概念重排
- 角色改用素材包角色 sprite，依家人照片調色

### 檢查點
先產靜態示意圖（新格局全景＋裝飾品項標示）給使用者確認，點頭後才接遊戲邏輯。

## 階段 4：裝飾定案
甜點、咖啡（蛋糕桌旁）、多處禮物盒——品項在階段 3 示意圖中標出供挑選，不另跑一輪。

## 不變更
搖桿、碰撞系統、對話流程、彩蛋、開場畫面邏輯全部沿用；只換渲染層與地圖資料。
