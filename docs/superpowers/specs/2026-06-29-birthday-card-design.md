# 爸爸生日卡片 — Design Spec
Date: 2026-06-29

## Overview
互動式像素風生日卡片網頁，以 Peeps 遊戲為風格參考。玩家操控爸爸角色在客廳中漫步，走近每位家人即可接收祝福。部署至 GitHub Pages（repo: `birthday-card-2026-dad`），目標裝置為手機直式瀏覽器。

---

## Technical Stack
- 純 vanilla HTML/CSS/JavaScript（無任何框架或函式庫）
- Canvas 2D 繪製場景與角色
- 單一 `index.html` 入口，資產依下方結構放置

```
birthday-card-2026-dad/
├── index.html
├── style.css
├── game.js
├── assets/
│   ├── tileset.png      # LimeZu Modern Interiors (CC-BY 4.0)
│   ├── characters.png   # LimeZu 角色 sprite，CSS 調色
│   └── bgm.mp3          # CC0 溫馨鋼琴 BGM，loop
└── docs/
    └── superpowers/specs/2026-06-29-birthday-card-design.md
```

## 角色規格

| 角色 | 位置 | Base Sprite 調色方向 |
|------|------|---------------------|
| 爸（玩家）| 中央起始 | 參考爸爸頭像：深色上衣、黑框眼鏡 |
| 媽 | 下方木頭沙發左側 | 參考媽媽頭像 |
| 哥 | 下方木頭沙發右側 | 參考哥哥頭像 |
| 弟 | 右側木頭沙發 | 參考弟弟頭像 |

- 頭像來源：`C:\for Claude\生日卡片\爸.jpeg` / 媽.jpeg / 哥.jpeg / 弟.jpeg
- 調色方式：Canvas `drawImage` + `globalCompositeOperation` 或 CSS `hue-rotate`
- 頭頂 HTML overlay 顯示名牌「爸/媽/哥/弟」

## 遊戲流程

### 1. 開場視窗
```
黑底覆蓋全畫面
╔═══════════════════╗
║  您已收到一份祝福🎉 ║
║                   ║
║      [查收]        ║
╚═══════════════════╝
```
點 [查收] → 0.5s 淡入過渡 → BGM 開始播 → 顯示客廳

### 2. 客廳探索
- 搖桿（左下浮動圓盤）操控爸爸上下左右移動
- 速度：3px/frame（約 60fps），家具格為碰撞障礙
- 接近 NPC（曼哈頓距離 ≤ 1.5 tile）→ 該 NPC 頭頂出現 💬 跳動動畫

### 3. 接收祝福
點 💬 → 彈出視窗：

```
╔══════════════════════════╗
║  來自媽媽的祝福 💓        ║
║──────────────────────────║
║  （祝福文字，可 scroll）  ║
║                          ║
║  [接收]          [關閉]  ║
╚══════════════════════════╝
```

- 文字 < 100 字：不 scroll
- 文字 ≥ 100 字：視窗內部加 `overflow-y: scroll`
- 點 [接收] 或 [關閉] → 💬 變為 ✓，視窗關閉

| NPC | 視窗標題 |
|-----|---------|
| 媽媽 | 來自媽媽的祝福 💓 |
| 哥哥 | 來自哥哥的祝福 🎁 |
| 弟弟 | 來自弟弟的祝福 🎂 |

### 4. 彩蛋（三個都讀完後）
1. 全螢幕彩色紙片撒花動畫（Canvas，60 顆紙片，3 秒）
2. 出現結尾視窗：
```
╔════════════════════════════════╗
║  🎂 你已收到全家人的祝福！     ║
║                                ║
║  [再接收一次祝福]  [完美收下]  ║
╚════════════════════════════════╝
```
- [再接收一次祝福]：3 個 💬 復活，視窗關閉
- [完美收下]：視窗關閉，撒花淡出，停在客廳
- 點完後右上角 🔄 始終可見，點擊重新整理頁面

## 音樂
- 來源：Pixabay 或 FreePD，CC0 授權，溫馨鋼琴風格，1–3 分鐘 loop
- 右上角 🔊/🔇 切換開關
- 手機禁止自動播放，因此綁在 [查收] 點擊事件觸發

---

## 搖桿實作
- 純 JS，約 50 行
- 固定圓底盤（80px）+ 浮動搖桿頭（30px）
- 觸控事件：`touchstart` / `touchmove` / `touchend`
- 輸出：方向向量，normalize 後乘以速度

---

## 碰撞系統
- 地圖以 16×16 tile 格子記錄障礙物（二維陣列 0/1）
- 爸爸的 hitbox：中心點 ± 6px
- 每 frame 分別測試 x 軸與 y 軸移動，獨立滑動

---

## 授權標註
- 畫面左下角極小字（10px，灰色，不透明度 50%）：`Tiles by LimeZu`
